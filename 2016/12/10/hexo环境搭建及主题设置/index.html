hexo环境搭建及主题设置

2016-12-10

今天把自己搭建hexo的过程记录一下，以后再搭建的时候可以少走些弯路。

  1. 下载安装Node.js和Git
    Node.js
    Gitbash
  2. 在某个文件夹下面建立hexo文件夹
  3. 进入文件夹，右键选择Git Bash Here的选项。然后在控制台中输入$ npm install -g hexo-cli。安装完成后，再输入命令 $ hexo init和$ npm install（注意顺序）。等待执行完成后，输入$ hexo g和$ hexo s。这时打开浏览器，在地址栏中输入localhost:4000预览效果。
  4. 创建Github账号，建立以yourname.github.io为名的仓库（将yourname替换成github账户的id）

  5. 更改hexo文件夹下的_config.yml文件
