Hadoop安装及配置

2016-06-01
Hadoop

这学期计算机体系结构课的老师为我们安排了有关Hadoop的实验。想要使用Hadoop，首先要配置Hadoop。在此，我把Hadoop安装及配置的过程写下，为自己留个底。

一、环境介绍

我是用虚拟机装的Hadoop，虚拟机的系统是Fedro20，JDK版本是1.8.0_92

二、Hadoop下载

我是从Hadoop的官网下载的Hadoop，网址是：http://hadoop.apache.org/releases.html

三、SSH免密配置

Hadoop是分布式的，各个主机之间通过SSH进行通信，就需要对SSH进行配置。我的系统自带了SSH，有的系统（如：Ubuntu）没有自带SSH，需要自行安装。因为我现在做的只是伪分布式，所以SSH的配置可能跟真实情况有所不同，这个以后再补充。

1、生成密钥对

[root@localhost ~]# ssh-keygen -t rsa

2、进入.ssh目录，复制密钥

[root@localhost ~]# cd .ssh
[root@localhost .ssh]# cp id_rsa.pub authorized_keys

3、测试

[root@localhost ~]# ssh localhost

四、安装Hadoop

这个部分主要的工作就是解压压缩包和修改环境变量

1、解压

[root@localhost ~]# tar zxvf /home/george/hadoop/hadoop-2.6.4.tar.gz -C /usr

2、修改Hadoop的环境变量

[root@localhost hadoop]# vi hadoop-env.sh
在hadoop-env.sh文件中加入
export JAVA_HOME=/usr/jdk1.8.0_92

3、修改系统的环境变量

[root@localhost hadoop]# vi /etc/profile
在文件尾部加入
export HADOOP_HOME=/usr/hadoop-2.6.4
export PATH=$PATH:$HADOOP_HOME/bin
保存之后，执行 souce /etc/profile 命令，使配置文件生效。

六、运行Hadoop

运行之前，需要执行命令 hadoop namenode -format，对系统文件进行格式化。之后，执行/sbin/start-all.sh，启动服务。启动服务后，执行 jps 命令，查看进程情况，若显示内容如下所示，则启动成功
[root@localhost sbin]# jps
4561 Jps
3825 SecondaryNameNode
4066 ResourceManager
4163 NodeManager
3512 DataNode
3359 NameNode

七、停止Hadoop

执行/sbin/stop-all.sh，停止服务
