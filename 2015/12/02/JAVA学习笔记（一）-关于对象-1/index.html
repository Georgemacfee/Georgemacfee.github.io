JAVA学习笔记（一）-关于对象(1)

2015-12-02
Java

一、啥是对象

众所周知，计算机不能完全的表达真实世界的每个细节。就像我最近在玩“战舰世界”，我只能操控船的一部分，而不能控制船的每一部分，有些部分是我操作不了的。这个把真实世界转化为计算机中模型的过程就被称为“抽象”。

就像《Java编程思想》中所说

所有编程语言都提供抽象机制

对象就是Java及其他面向对象的语言对真实世界进行抽象的方法。面向对象的语言具有五个基本特性

1.万物皆对象
这个世界上所有的事物都可以转化为对象，你是对象，我是对象，大家都是对象。没有什么东西是不能转化为对象。

2.程序是对象的集合，它们通过发送信息来告知彼此要做的

3.每个对象都有自己的由其他对象所构成的存储
在构成一个新类型的对象的时候，其可以包含其他类型的对象

4.每个对象都拥有其类型
对象是不能单独存在的。对象的存在必须依托类型，每个对象都有自己的类型

5.某一特定类型的所有对象可以接受同样的信息
举个例子，“老虎”和“狮子”都是“食肉动物”，那么，它们可以接受同一类型对象“肉”。

三、访问控制

对于计算机来说，安全是很重要的。有些属性是我们不想要其他人知道的。我们需要建立一个边界。Java中提供了三个关键字public,private,protected。public是任何人都可以直接访问的。private是只有类型创建者和类型内部的方法可以访问。而protected除了类型的创建者和类内方法外，其继承类也可以访问。

五、类的继承

当我们学习其他课程的时候，经常会出现一个概念被包含于其他概念的情况，比如 长方形、菱形 都是被包含于 平行四边形 的。这时候，我们很自然地认为 长方形、菱形 具有 平行四边形 的性质。

在编写程序时，我们有时也会遇到一种情况：新的类型具有已有类型的所有属性和方法，但也有新的属性和方法。 这时，如果我们重新编写新的类型的话会比较浪费时间和空间。所以，就有了 继承 来解决这一问题。

就拿动物来举个例子：

在上面的类型中， 动物 可以吃饭、睡觉、移动。当然，人 也 可以 吃饭、睡觉、移动。除此之外，人 还 可以玩电脑，看电视。前三项技能是所有动物都会的，人也会，所以直接从 Animal 继承就好。 而后两项技能只有人才有，所以Human 类又新加了这个方法。
